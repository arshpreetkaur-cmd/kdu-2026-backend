# Exercise 1: Personal Digital Bookshelf (REST Fundamentals)

## Overview
This project implements a simple RESTful API using Spring Boot that allows users to manage a personal book collection.  
It demonstrates core REST principles such as correct HTTP methods, resource-based URLs, and proper HTTP status codes.

---

## Tech Stack
- Java
- Spring Boot
- Spring Web
- Maven
- In-memory storage

---

## Base URL
http://localhost:8080/api/books
