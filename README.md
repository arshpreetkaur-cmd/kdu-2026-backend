# Warehouse Inventory Search and Tracking System

This project implements a simple **warehouse inventory search and tracking system** using core Java concepts.  
It focuses on searching, filtering, and tracking inventory data in an efficient and readable way.

---

## Description

The Warehouse Inventory Search and Tracking System allows users to:
- Store inventory items in memory
- Search items based on different criteria
- Track item availability and quantities
- Display filtered inventory results

This project is designed to demonstrate **clean Java logic and data processing**, not database integration.

---

## Features
- Search inventory items by name or category
- Filter items based on quantity or availability
- Display inventory details in a readable format
- Simple and modular code structure

## Question 3: Configuration Constraint System (Annotations & Validation)

### Description
This module implements a **custom validation system** using Java annotations and reflection.

### Key Features
- Custom annotation `@RangeCheck` with min/max constraints
- Runtime validation using reflection
- Custom unchecked exception for validation failures

### Components
- `@RangeCheck` annotation
- `ConfigValidator` validation engine
- `ConfigValidationException`
- `SystemConfig` configuration class
